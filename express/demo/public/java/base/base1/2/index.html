# 1.什么是变量：用来保存运算时数据的一小段内存空间

  • 变量名:由于内存地址不好记，给变量起的别名
  • 变量类型：内存空间类型
  • 变量值：变量中存储的数据

# 2.变量命名规范：

  • 变量名必须以字母或下划线或$符号打头，名字中间只能由字母、数字和下划线“_”组成；
  • 简短且能清楚地表明变量的作用
  • 通常第一个单词的首字母小写，其后每个单词的首字母大写；
  • 变量名不能是JAVA中的保留字或（关键字）。
  • 关键：字、下、美、元、数、骆驼。

String：字符串 随便 String stuName="张三";

# *4.运算符

运算符 描述 实例
+ 加 10 + 20 = 30
- 减 10 - 20 = -10
* 乘 10 * 20 = 200
/ 除 10 / 20 = 0.5
% 取余数 返回除法的余数 9 % 2 = 1
= 赋值运算符 int score =100;
> , < , >= , <= , == ,!= 关系运算符 a>=10
boolean类型表达式 ? 条件成立的结果1 : 条件不成立的结果2 三元运算符 c = a>b?10:20
  • && || ! 逻辑运算符

&&: 两个都为true 则为true
||： 一个为true 则为true
!: 如果是true 则为false 反之亦然

else:否则

  • # 单个if ：

语法：

 if(boolean类型条件){  
      条件成立的语句 
   }

# 6、switch语句

语法：

switch (int/char/short/byte/String/enum/long表达式) {

 	case 常量 1:
 		语句;
 	break;
 	case 常量 2:
 		语句;
 	break;
 	   …
 	default:
 	语句;
 	break;
   }

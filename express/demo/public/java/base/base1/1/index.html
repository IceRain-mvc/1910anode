# 第1章

# 0、java：Java是一门面向对象编程语言

# 1、什么是程序：

  • 为了让计算机执行某些操作或解决某些问题而编写的一系列有序指令的集合！

# *2、java技术平台

JAVAEE 平台企业版
JAVASE 平台标准版
JAVAME 平台移动/微型版

# *3、配置环境变量

  • 帮助地址：https://jingyan.baidu.com/article/6dad5075d1dc40a123e36ea3.html 在系统变量中增加

a、新建 JAVA_HOME 变量 ，变量值填写jdk的安装目录 例如： C:\Program Files\Java\jdk1.8.0_161

b、编辑Path 变量，在变量值最后输入 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

c、新建 CLASSPATH 变量，变量值填写 .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意最前面有一点）

# 4、JDK

JDK(Java Development Kit) 是 Java 语言的软件开发工具包

# *5、用记事本开发java程序步骤

1、编写源程序
2、编译源程序(翻译成字节码文件[.class],命令是 javac 文件名.java )
3、运行 (命令是 java 文件名)

javac: 编译（把程序员写的java代码编译成二进制文件.class） 语法： javac 文件名.java java:运行（让java虚拟机运行二进制文件） 语法：java 文件名

# *7、转义符

  • *\:代表转义符号，目的是转义后面紧跟的字母
  • *\\:代表输出一个 \
  • *\n:代表换行
  • *\t:代表tab（8个空格）

# *8、java注释

  • 单行注释（//） 一般放在行上方或行尾部
  • 多行注释（以/*开头 */结尾） 写思路或步骤，可以放在任意位置
  • 文档注释（以/**开头 */结尾） 方法或类的说明，必须放在方法、属性、类上面

# 9、Eclipse 开发java程序的步骤

  1. 创建一个java project
  2. 创建一个类，写上包名（全小写），写上类名（每个单词首字母都大写）
  3. 编译运行

# 10、集成开发环境 简称 IDE ：集合了开发环境和调试功能，帮助程序员开发软件。

# 11、Eclipse快捷键

提示：alt + / 单行注释和取消注释快捷键： ctrl+/ 格式化快捷键：Ctrl+Shift+F 文档注释快捷键：alt+shift+j

# 1、XML（EXtensible Markup Language）,可扩展标记语言

# 2、XML特点

  • XML与操作系统、编程语言的开发平台无关
  • 实现不同系统之间的数据交换

# 3、XML作用

  • 数据交互
  • 配置应用程序和网站
  • Ajax基石

# 4、XML编写注意事项

  • 所有XML元素都必须有结束标签
  • XML标签对大小写敏感
  • XML必须正确的嵌套
  • 同级标签以缩进对齐
  • 元素名称可以包含字母、数字或其他的字符
  • 元素名称不能以数字或者标点符号开始
  • 元素名称中不能含空格

# 5、XML解析器类型

非验证解析器

  • 检查文档格式是否良好

验证解析器

  • 使用DTD检查文档的有效性

# 6、XML命名空间的作用

  • 解决在复杂、大型XML文件中，出现名称相同，但是含义不同的元素

# 7、解析XML技术

名称 原理 优点&适用范围 特点
DOM 基于XML文档树结构的解析 适用于多次访问的XML文档 特点：比较消耗资源
SAX 基于事件的解析 适用于大数据量的XML文档 特点：占用资源少，内存消耗小
DOM4J 非常优秀的Java XML API 性能优异、功能强大 开放源代码

# 8、DOM4J的常用类

  • Document：定义XML文档
  • Element：定义XML元素
  • Text：定义XML文本节点
  • Attribute：定义了XML 的属性

# 9、DOM4J的示例

public static void main(String[] args) {
	try {
		SAXReader reader = new SAXReader();// 创建一个能够读取xml文件的对象
		Document document = reader.read(new File("src/scores.xml"));// 建立了一个和xml文件关联的document对象
		Element scores = document.getRootElement();// 获取xml的根节点scores
		for (Object obj : scores.elements()) {//scores.elements():获取当前结点的所有子节点
				Element student = (Element) obj;
				//elementText("name")获取指定名称子节点的text文本
				System.out.println(student.elementText("name") + "  "
				+ student.elementText("yjscore") + "  "
						+ student.elementText("sjscore"));
			}
		
	} catch (Exception e) {
		e.printStackTrace();
	}
}

# 1、为什么要用枚举

减少代码量 方便记忆 方便对连续数据的修改

# 2、枚举的好处

  • a、类型安全
  • b、解决意义不明确
  • c、代码更优雅
  • d、定义自己的类型

	Genders sex=Genders.Male;

# 5、字符串处理方法

方法 含义
int 字符串.length() 获取字符串的长度，返回一个int类型的结果(包含空格)
boolean 字符串1.equals(字符串2) 判断字符串是否完全一致 （区分大小写）
boolean 字符串1.equalsIgnoreCase(字符串2) 判断字符串在不区分大小写的前提下是否一致
String 字符串.toUpperCase() 将字符串中的所有内容转换成大写形式(不包括数字)
String 字符串.toLowerCase() 将字符串中的所有内容转换成小写形式(不包括数字)
int 字符串1.indexOf(要查找的字符串) 返回要查找的字符串在前面字符串1出现的位置（从0开始，没找到返回-1,包含字符串出现的位置）
int 字符串1.indexOf(要查找的字符串,起始位置) 返回要查找的字符在前面字符串1出现的位置（从起始位置开始，没找到返回-1,包含字符串出现的位置）
int 字符串.lastIndexOf(要查找的字符串) 返回要查找的字符串在前面字符串1最后出现的位置（从0开始，没找到返回-1,包含字符串出现的位置）
String 字符串1.substring(开始的位置/下标) 返回从下标开始之后的所有字符串（包含下标位置）
String 字符串1.substring(开始的位置/下标,结束的位置/下标) 返回从下标开始到结束之间的所有字符串（包含开始下标位置，不包含结束下标位置）
String 前后包含空格的字符串.trim() 返回去掉前后空格的新字符串(字符中间的空格不受影响)
String 字符串1.replace(旧的字符串, 新的字符串) 把所有旧的字符串替换成新的字符串
String[] 含有特殊字符的字符串.split(要分割的字符) 按照指定的字符进行分割，返回分割后的字符串数组
Char[]a=String.toCharArray() 把数组转换成字符数组的方法

# 10、Calendar 日期处理类

	Calendar rightNow = Calendar.getInstance();//getInstance()创建一个带有当前时间的Calendar对象
	rightNow.add(Calendar.MONTH, -4); //add(日历字段,时间量)  为给定的日历字段添加或减去指定的时间量。
	Date date=rightNow.getTime();  //获取当前时间
	SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
	System.out.println(sdf.format(date));
	System.out.println("----------------------");
	Calendar curr = Calendar.getInstance();
	System.out.println(curr.get(Calendar.YEAR));//年
	System.out.println(curr.get(Calendar.MONTH)+1);//当前月
	System.out.println(curr.get(Calendar.DAY_OF_YEAR));//一年中的第几天 日
	System.out.println(curr.get(Calendar.HOUR));//获取当前的小时
	System.out.println(curr.get(Calendar.MINUTE));//获取当前的分钟
	System.out.println(curr.get(Calendar.SECOND));//获取当前的秒
	System.out.println(curr.get(Calendar.DATE));//本月第几天
	System.out.println(curr.get(Calendar.DAY_OF_WEEK_IN_MONTH) );//本月的第几周
	System.out.println(curr.get(Calendar.DAY_OF_WEEK)); //本周的第几天

# 特别注意:

一个网页就好像是一栋房子，HTML结构就是一面面的墙，而标签则是一块块的砖，砖要摆放有序，整整齐齐，墙才会牢固。

最后css则是装饰材料，美不美就靠她了。 因此，我们要有一个优秀的网页，

既要提供一个干净而又有清晰结构的HTML，更加离不开css的极致美化。

# 11、超链接的3种语法

  • 页面链接:<a href=”链接地址” target=”目标窗口（_self/_blank）”>链接文字或图像</a>。
  • 锚链接：
    • 锚点写法：<a name=”xxx”></a>
    • 跳转写法：<a href=”#xxx”>回到位置</a>
    • 跨页面跳转写法：<a href=”页面名称#xxx”>回到位置</a>
  • 功能性链接：<a href=”mailto:xxxx”>联系我们</a>这个死写法。

# 18、在设置CSS样式的语法，在里定义从结束。

  • 标签选择器：选择器的名字都是html标签,所有出现的标签全都改变样式
    • 例如：
    p{font-size:36px;}
1
  • 类选择器：选择器的名字是自定义的但是必须要有"." ，需要用到的时候使用 class属性调用
    • 例如：
    .java{font-size:36px;}


    <h5 class="java">旦法师大旦法师大旦法师大旦法师大</h5>
1
2
3
4

  • ID选择器：选择器的名字是自定义的但是必须要有"#" ，需要用到的时候使用id属性调用,

    注意：由于标签的ID属性在同一个HTML页面中只能出现一次(标签id不能相同)，

    这也就决定了同名的ID选择器只能调用一次。

    • 例如：

    <head>
        <style type="text/css"> 
        #java{font-size:36px;} 
        </style>
    </head>
    
    <h5 id="java">旦法师大旦法师大旦法师大旦法师大</h5>

# 19、复合选择器

  • 交集选择器：开头第一个必须是标签选择器，第二必须是类选择器或者ID选择器。
    • 俩个标签不准有空格，必须连续书写。
    • 示例：

    p.ni{ color：bl font-size：16px；}
    <p class=“ni”（不可以少任何一个）>….里面字体全为蓝色..字体16px</p>。
1
2
3

  • 后代选择器：按标签嵌套写样式，可以嵌套各种标签，但是定义了多少标签就要写多少标签, 要不然不能使用。p em strong{color：red；}每个用空格隔开。

  • 并集选择器：

h3，em，p，.ta{color：green；  font-size：16px；}
1

在调用时，只要有其中意一个标签就行,

  • 比如<em>…</em>里面的文字为绿色，大小为16px。用逗号隔开每个标签。

# 21、引入CSS样式的三种方式

# 行内样式：开始标签中加入style属性

  • 例如：
<h1 style="color:red;">一级标题</h1>
1

# 内部样式：在head标签中加入style标签

  • 例如：

 <head><style type="text/css"> .top{font-size:36px;} </style></head>
1
2

# 外部样式：新建css文件，在head标签中加入link标签引用css文件

  • 例如

<head><link href="aaaa.css" rel="stylesheet" type="text/css" /></head>
